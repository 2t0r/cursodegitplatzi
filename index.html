Comandos utiles de la terminal

clase 11

cd : Moverse entre carpetas

mkdir : crear carpetas

ls : lista de archivos en una carpeta

touch : Crear un archivo

rm [Nombre-archivo] : Borra el archivo

rm -rf [Nombre-carpeta] : Borra carpeta

clear : limpia la terminal

Creando Repositorios

clase 12

git init : nos crea un repositorio de manera local y lo hará en la carpeta donde estamos posicionados o se le puede pasar [nombre_de_la_carpeta] y creará la carpeta con ese nombre.

Los repositorios se crean como carpetas oculas, para eliminarlos puedes usar el explorardor de archivos o rm-rf [nombre del repo]

git add | rm | status (Agregando, quitando y viendo el estatus de archivos)

clase 13

git add [Nombre archivo] Agraga el archivo al staging

git add -A Agraga todos los archivos pendientes al staging

git status Para ver en que estado se encuentran los archivos

git rm --cached :Nos devolvemos un paso

git rm -f [file] :Elimina el archivo por completo.

git add -n [file] : Que no haga nada

git commit (Confirmando cambios)

clase 14

Git commit -m [“mensaje”] : Confirma cambios. Se recomienda ser descriptivo en el mensaje para tener trazabilidad

Git commit -m [“mensaje”] -amend : agregar nuevos cambios al ultimo commit

git log : Nos muestra la historia de todos los commits que hemos realizados.

git tag (Etiquetando confirmaciones)

clase 15

git tag -a -m"": Nos permite agregar etiquetas a nuestros cambios. para agregar tag a otro commit se usa el hash

git tag -l: Nos muestra la lista de etiquetas.

git tag -f [hash]: Para renombrar. Se coloca el hash

git tag -d : Para borar.

Git commit -m [“mensaje”] -amend : agregar nuevos cambios al ultimo commit

git log : Nos muestra la historia de todos los commits que hemos realizados.
